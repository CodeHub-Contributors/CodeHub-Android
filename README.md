## CodeHub-Android

CodeHub is a Persian pastebin service that provides all kind of software services completely Free and Open Source. There are a numores repositories for those who want to create their own CodeHub.

CodeHub-Android is the Android application of CodeHub which provides several services. It's 100% Free and designed responsively. In this version, you have access to create snippets in the following languages.
* Java
* Python
* PHP
* HTML
* Go

### B4X
B4X is a suite of rapid application development IDEs and proprietary programming language that allows the creation of applications on the following platforms: Google Android, Apple iOS, Java, Raspberry Pi and Arduino. Although the B4X syntax is very similar to BASIC, it is an entirely new language.
